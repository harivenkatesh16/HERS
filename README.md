# HERS E-commerce Website

A modern, responsive starter storefront for the **HERS** brand.

## Features
- Hero section and brand messaging
- Product grid with add-to-cart interactions
- Sticky navigation
- Newsletter sign-up form
- Fully static (HTML/CSS/JS), easy to deploy on any static host

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`.
